# Security Guidelines

## 禁止提交敏感配置

- 禁止提交真实配置文件、密钥、令牌、Cookie、账号密码。
- 所有本地配置必须以 `*.example` 形式入库。
- 真实配置文件应通过 `.gitignore` 忽略。

## 数据与上下文边界

以下内容禁止进入仓库与模型上下文：
- 内网信息（如真实 IP、私有域名、内部网络拓扑）
- 访问凭据（token、cookie、AK/SK、会话信息）
- 个人信息（手机号、邮箱、身份证号等）
- 客户数据与业务敏感数据
